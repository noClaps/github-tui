# GitHub TUI

A simple command line tool to browse GitHub, built using Go and [Charm](https://charm.sh).

## Build instructions

1. Clone the repository.

   ```sh
   git clone https://github.com/noClaps/github-tui.git
   cd github-tui
   ```

2. Build the package, and run it.

   ```sh
   go build
   ./github-tui
   ```
